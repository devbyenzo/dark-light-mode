# 🌗 Dark & Light Mode Page

Página simples desenvolvida utilizando **HTML**, **CSS** e **JavaScript** com sistema funcional de alternância entre **modo claro** e **modo escuro**.

O projeto foi criado para estudos de manipulação do DOM, estilização dinâmica e interação com o usuário.

---

# 📌 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

# 🧠 Como Funciona

O sistema funciona utilizando JavaScript para:

1. Detectar o clique do usuário
2. Alternar classes CSS
3. Modificar automaticamente as cores da interface

---

# ⚙️ Funcionamento do Tema

O JavaScript utiliza:

```javascript id="wr9dfc"
classList.toggle()
```

para alternar entre:

* Tema claro
* Tema escuro
